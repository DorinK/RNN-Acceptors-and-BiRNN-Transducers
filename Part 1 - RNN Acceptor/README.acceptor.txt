README for experiment.py:

In the current working directory should be:
1. experiment.py
2. gen_examples.py


One parameter to experiment.py:
1. The chosen language (regular/palindrome/duplicate/cross)


For example in the command line you should enter: 
python3 experiment.py regular
python3 experiment.py palindrome
python3 experiment.py duplicate
python3 experiment.py cross


* The generated train and test files of the chosen language will outputted to the CWD.
